# Coches-mod
Aquí podréis encontrar vehículos moddeados.

🚗💸 COCHES MODDEADOS GTA ONLINE 💸🚗

📥 1. Descarga el vehículo
Dentro del presente repositorio verás varias carpetas 📂
En cada carpeta hay:
📸 Una imagen del coche (preview)
📄 Un archivo .json --> tienes completa libertad para cambiar el nombre del archvio .json al que más te guste o al que te permita identificarlo mejor en los siguientes pasos (IMPORTANTE)

👉 Descarga solo el archivo .json

📁 2. Guarda el archivo en Cherax

Sigue esta ruta en tu PC 🖥️:

➡️ Tecla Windows
➡️ Documentos
➡️ Cherax
➡️ Vehicles 📌 Pega ahí el archivo .json



🎮 3. Cargar el coche en el juego

Dentro de GTA con el mod menú:

➡️ Ve a Spawner (icono garaje 🚪)
➡️ Baja hasta "Saved Vehicles" 
➡️ Selecciona tu coche
➡️ Pulsa Load Vehicle

🔥 ¡Listo! Ya puedes usarlo

💾 4. Guardarlo como tuyo (MUY IMPORTANTE)

Si quieres quedártelo permanentemente:

➡️ En Spawner
➡️ Busca "Save current vehicle as PV" 
➡️ Elige el garaje 🏠

💸 Ahora el coche será tuyo en el juego
